## McCabe 度量法
McCabe 度量法是一种基于程度控制流的复杂度量方法，该方法认为程序的复杂性很大程度上取决于控制的复杂性。首先根据程序画出程序图，然后基于图论用图的环路数来度量程序的复杂性，即 V(G)=m-n+2p ,其中 m、n 和 p 分别表示图 G 中弧的个数、顶点的个数和强连通分量数。由于程序转化出来的程序图正常情况下强连通分量数都是 1，所以可以简化为 V(G)=m-n+2。
